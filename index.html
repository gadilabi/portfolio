Project-4: Video and Text Chat App

Links

Live site: https://letsstarttalking.herokuapp.com/

Description

This little website allows users to create rooms where they can chat with their friends. Users can either send messages to the entire room or private messages to specific users. Moreover, Video calls are allowed between any two users.

Technologies in use

The backend uses Node.js (with express). The frontend is built with web components. The text chat is achieved with websockets (with the help of the socket.io library). The video chat is achieved through the use of WebRTC. I used the native API , without the use of any libraries like Peer.js.
